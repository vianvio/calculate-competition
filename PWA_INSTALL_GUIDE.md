# PWA 安装指南

本应用已支持 PWA（Progressive Web App），可以像原生应用一样安装到设备主屏幕。

## 功能特性

✅ 离线访问支持
✅ 快速加载
✅ 主屏幕图标
✅ 全屏体验
✅ 自动更新

## 安装方法

### 在 iPad/iPhone (Safari)

1. 打开 Safari 浏览器访问应用
2. 点击首页的 **📱 安装到主屏幕** 按钮
3. 或者点击底部的分享按钮 (⬆️)
4. 选择 "添加到主屏幕"
5. 确认并完成安装

### 在 Android (Chrome)

1. 打开 Chrome 浏览器访问应用
2. 点击首页的 **📱 安装到主屏幕** 按钮
3. 或者点击浏览器菜单（三个点）
4. 选择 "安装应用" 或 "添加到主屏幕"
5. 确认并完成安装

### 在 Windows/Mac (Chrome/Edge)

1. 打开 Chrome 或 Edge 浏览器访问应用
2. 点击首页的 **📱 安装到主屏幕** 按钮
3. 或者在地址栏右侧找到安装图标 ⊕
4. 点击后确认安装
5. 应用将作为独立窗口打开

## 安装后的优势

- **更快的启动速度**：应用资源已缓存到本地
- **离线使用**：即使没有网络也能使用基本功能
- **沉浸式体验**：全屏显示，无浏览器地址栏
- **桌面图标**：像原生应用一样从主屏幕启动
- **自动更新**：后台自动更新到最新版本

## 卸载方法

### iOS
长按应用图标 → 删除应用

### Android
长按应用图标 → 卸载或应用信息 → 卸载

### Windows/Mac
右键应用窗口 → 卸载
